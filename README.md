# Projekt Feelique - Webtechnologien

### Projektbeschreibung
In diesem Projekt geht es um die Entwicklung einer Webanwendung namens "Feelique", die es Benutzern ermöglicht, ihre Emotionen zu verfolgen und zu analysieren. Die Anwendung bietet Funktionen wie das Hinzufügen von Emotionseinträgen, das Anzeigen von Statistiken und das Anpassen von Einstellungen.

### M1 
- ***Entity-Klasse*** `MoodEntry` (definiert Stimmung, Intensität und Zeitpunkt)
- **GET-Route:** `/moods` (liefert eine Liste von Beispiel-Stimmungseinträgen)
